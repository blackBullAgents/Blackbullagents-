# Chapter 7 — Flexible Fee Routing

## Overview

Building on Chapter 3, this chapter goes deeper into how routing percentages can be adjusted over time and how those changes show up in the dashboard.

## Adjusting Routing Mid-Flight

Teams can change their routing percentage without disconnecting or re-authorizing from scratch. Changes apply from the next detected fee event onward.

## Monitoring Impact

The dashboard surfaces a before/after view so teams can correlate routing changes with subsequent liquidity engine activity.
