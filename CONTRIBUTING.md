# Contributing to BlackBull Agent

Thanks for your interest in contributing! Here's how to get started.

## How to Contribute

1. **Fork** this repository
2. **Create** your branch: `git checkout -b feat/your-contribution`
3. **Commit** your changes: `git commit -m 'feat: add X'`
4. **Push** to your fork: `git push origin feat/your-contribution`
5. **Open a Pull Request**

## Reporting Issues

- 🐛 Found a bug? [Open an issue](../../issues/new)
- 💡 Have a feature idea? Start a discussion

## Code Style

Keep changes focused and well-described. Include context in your PR description so reviewers understand the "why," not just the "what."

## Code of Conduct

By participating, you agree to uphold our [Code of Conduct](CODE_OF_CONDUCT.md).
