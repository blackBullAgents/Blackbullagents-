# Chapter 11 — Frequently Asked Questions

**Do I need to give up custody of my token?**
No — the system is designed to be non-custodial. See Chapter 8.

**Can I turn off fee routing later?**
Yes, routing permissions are revocable at any time from the dashboard. See Chapter 3 and Chapter 9.

**What happens if the engine misbehaves?**
Use the kill switch (Chapter 9) to immediately halt activity.

**Which launchpads are supported today?**
Meteora, Pump.fun, and Kickstart. See Chapter 4.

**I'm not ready to connect a wallet — can I still explore?**
Yes, the dashboard and documentation can be reviewed without connecting a wallet.
